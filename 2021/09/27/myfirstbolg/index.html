Markdown简介

Markdown 是一种轻量级标记语言，能将文本转换成有效的XHTML（或者HTML）文档。
markdown语法简洁明了易于掌握，能够良好适用于各大CMS。（csdn也支持使用markdown语法书写博客）
markdown让HTML文档变得易读、易写、易改，但它并不能取代HTML，HTML是一切网页的基础，markdown语法种类很少，只对应HTML标记的一小部分。
HTML是一种发布格式，Markdown是一种书写格式。

超链接

  • 行内式

    [链接标题](链接地址)
    比如欢迎来到我[杂乱无章的小窝](https://dinghongtong.github.io)

效果：比如欢迎来到我杂乱无章的小窝

  • 参考式

万一一行里有一堆超链接咧，地址全写在一起大大降低了代码可读性，markdown语法目的不就是易读易写易改吗；

这一行有 [一][1]、 [二] [2]、 [三] [3]、 [四] [4]个链接

[1] :https://dinghongtong.github.io “一”
[2] :https://dinghongtong.github.io “二”
[3] :https://dinghongtong.github.io “三”
[4] :https://dinghongtong.github.io “四”

效果：
这一行有一、 二、 三、 四个链接

  • 脚注

和参考式其实是一个东西，内容形式有点区别

这是一个链接不知名的小站的[脚注1]。

[脚注1]: https://dinghongtong.github.io

这是一个链接不知名的小站的^脚注。

  • 你也可以直接写一个地址，会被自动识别为链接
    或者规范一点用个半角符号括起来
    https://dinghongtong.github.io
    <https://dinghongtong.github.io)>
    https://dinghongtong.github.io

注意：无论哪种情况链接标题和链接地址之间都不能有空格

引用

如果不是复制粘贴的话我一般在每一行开头用 > 符号把代码当成引用
偷懒只在一个段落首行打>也可以,记得在段落最后换行

示例：

>#include <stdio.h>
>int main(void)
>{
> printf(“Hello world\n”);
>}

效果：

#include <stdio.h>
int main(void)
{
printf(“Hello world\n”);
}

引用可以嵌套

多打几个 > 就可以

表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

|内容|内容|内容|
|-|- -|- - -| (小横数目无所谓，有无空格无所谓，但是一定要有表头，即|-|-|-|)
|内容|内容|内容|
|内容|内容|内容|

示例：

这个位置一定要留一个空行，不然不会生成表格

x o x
o o x
x x o

这时一定会有强迫症朋友想把它对齐（比如我）

对齐方式：

| 左对齐 | 右对齐 | 居中对齐 |
| :-| -: | :-: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

实例如下：

左对齐 右对齐 居中对齐
单元格 单元格 单元格
单元格 单元格 单元格

很抱歉，markdown语法目前不支持这一点，但是你可以在里面嵌套HTML、CSS，同理你可以用HTML代码做到合并单元格等操作。

当锚点是标题

锚点是内容

<a id=”idname”>内容>
